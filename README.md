# Blood Bank Management System (Pro)

Full-stack Flask project with authentication and tested manual, API , automation testing donor management, and search system.

## Features
- Login & Register
- Add donors
- Search blood group
- Dashboard

## Run
pip install -r requirements.txt
python app.py
